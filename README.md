# Nile Gazer · Full-Stack Developer Portfolio

**Live site:** [nilegazer00.github.io](https://nilegazer00.github.io)

A high-performance, conversion-focused static site built to attract clients, demonstrate technical expertise, and provide free utility tools for developers and business owners.

---

## 🚀 Features

- **Lead generation system** – Project estimate form (Web3Forms) + contact form (Formspree)
- **3 interactive tools** – Freelance Rate Calculator, Web Dev ROI Estimator, Project Timeline Planner
- **Gated content strategy** – Tools unlock detailed guides, driving email capture
- **Mobile‑responsive design** – Works seamlessly on all devices
- **No backend required** – Fully static, deployable on GitHub Pages
- **Google Analytics 4 integrated** – Track visitor behavior and conversions

---

## 🛠️ Tech Stack

| Layer       | Technologies |
|-------------|--------------|
| Frontend    | HTML5, CSS3, Vanilla JavaScript |
| Forms       | Web3Forms (lead capture), Formspree (contact) |
| Analytics   | Google Analytics 4 (gtag.js) |
| Hosting     | GitHub Pages |
| Version control | Git |
